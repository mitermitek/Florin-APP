# Florin APP

Application de gestion financière personnelle construite avec Angular et Angular Material.

## Technologies

- Angular 21.0.0
- Angular Material 21.0.6
- TypeScript 5.9.2
- RxJS 7.8.0
- Vitest 4.0.8

## Installation

```bash
npm install
```

## Démarrage

```bash
npm run start
```

L'application sera accessible sur `http://localhost:4200/`

## Scripts disponibles

- `npm run start` - Démarre le serveur de développement
- `npm run test` - Lance les tests unitaires
- `npm run build` - Build de production
- `npm run watch` - Build en mode watch
